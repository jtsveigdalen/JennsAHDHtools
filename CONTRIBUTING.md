# Contributing

Thanks for helping improve this project.

## Scope

This repo contains small ADHD-friendly tools.

Please keep changes:

- Small and focused
- Easy to understand
- Dependency-light (or dependency-free when possible)

## How to contribute

1. Fork the repo
2. Create a branch for your change
3. Make the change
4. Test manually in a browser
5. Open a pull request with a short description

## Pull request checklist

- [ ] Change is limited to one clear purpose
- [ ] UI text is clear and concise
- [ ] No unrelated refactors
- [ ] Manual test notes are included in PR description
- [ ] Docs updated if behavior changed
- [ ] Note any accessiblity features, etc.  

## Commit messages

Use clear, short commit messages, for example:

- `fix: correct export file handling`
- `feat: improve score panel readability`
- `docs: add usage notes`

## Reporting bugs

Please include:

- Browser and version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshot (if helpful)
